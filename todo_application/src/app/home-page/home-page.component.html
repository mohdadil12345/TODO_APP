• The Todo Application is a web-based task management tool built using Angular. It allows users to create, edit, delete, and search todo items. The application provides offline support using local storage and online synchronization with a Firebase backend.

  • Create new todo items with a title, description, and status.
  • Edit existing todo items including title, description, and status.
  • Delete todo items.
  • Search todo items by title.
  • Filter todo items by status
  • Offline support: Ability to perform CRUD operations when offline, with synchronization when online.
  • Multilingual support: Ability to change the application language (English, French, Spanish, Tamil).

  • Manages the main todo functionality including adding, editing, deleting, and filtering todo items.
  • Displays the list of todo items.
  • Utilizes CKEditor for rich text editing.
  • Implements offline support and synchronization with Firebase.

  • Provides a popup window for editing todo items.
  • Utilizes CKEditor for editing todo item details.
  • Emits events to update todo items.

  • CKEditor: Integrated for rich text editing capabilities in todo item descriptions.
  • ngx-translate: Used for multilingual support, allowing language selection and translation of application texts.
  • Toaster: Provides toast notifications for various actions such as adding, editing, and deleting todo items.

  • Clone the repository to your local machine.
  • Install dependencies using npm install
  • Run the application using ng serve
  • Access the application in your browser at http://localhost:4200.
